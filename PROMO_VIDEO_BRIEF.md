# PlotRail Promo Video Brief

Format: 1080x1920 vertical MP4, silent, designed for Douyin and Xiaohongshu background music.

## Narrative Arc

1. AI writes one good scene.
2. Long fiction breaks: canon, character state, secrets, and plot threads drift.
3. PlotRail turns the novel into files the agent can inspect.
4. The author gives one prompt.
5. The agent forms a writing-room workflow: canon keeper, chapter planner, continuity reviewer, author-style analyst.
6. Every chapter moves through contract, draft, review, memory update.
7. The result is not "AI writes for you"; it is "AI stays on the rails while you direct."

## Prompt To Show

```text
Use the novel-writer skill.
Initialize this folder as a long-form novel project.
Do not write prose yet.
First generate 3 proposals from my theme, target readers,
taboo content, style references, and ending direction.
Keep proposals outside canon until I approve one.
```
