## What This Case Proves

PlotRail is not just a prompt template. It creates a repeatable workflow:

1. turn an idea into approved story state
2. create a chapter contract before prose
3. draft against that contract
4. review continuity, character motivation, secrets, hooks, and state changes
5. learn from the author's revisions

The strongest extracted author preference:

```text
Do not let the protagonist win because the plot needs him to win.
Show the procedure, constraint, test, fallback, and cost.
```
